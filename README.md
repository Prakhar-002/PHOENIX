# Phoenix AI Features 🍂🧸

Phoenix AI is an **AI-powered chatbot** that not only generates intelligent responses but also **stores and organizes conversations** efficiently. Below are its key **use cases**:  

## **AI-Powered Conversations 🌍**  

- `Smart Responses` 🤖 Get **real-time, AI-generated** replies powered by **Gemini API**.  
- `Context Awareness` 🧠 Phoenix AI remembers **previous chats** for meaningful interactions.  
- `Markdown Support` 📝 Format messages with **bold, italic, and code snippets**.  
- `Syntax Highlighting` 🖥️ AI-generated code is displayed beautifully with **React Syntax Highlighter**.  

<br/>

## **Conversation Management 💾**  

- `Auto-Generated Titles` 🏷️ AI automatically **creates titles** for saved conversations.  
- `Chat History` 📜 View **past interactions** anytime in an **organized interface**.  
- `Real-Time Storage` ⚡ Conversations are stored securely in **Appwrite Database**.  

<br/>

## **User Authentication 🔐 & Security 👮🏻‍♂️**  

- `Sign Up & Log In` 🔑 Secure **user authentication** powered by **Appwrite**.  
- `Password Reset` ✉️ Reset your password via a **secure email link**.  
- `Session Management` 🔄 Stay logged in with **Appwrite's secure session handling**.  

<br/>

## **User Experience 🖌️ & UI Enhancements 🎨**  

- `Smooth Animations` 🎭 Powered by **Framer Motion** for an interactive feel.  
- `Real-Time Navigation` ⚡ Instant page transitions with **React Router DOM**.  
- `Customizable UI` 🎨 Modify styles easily using **TailwindCSS**.  

<br/>

## **Deployment 🦾 & Scalability ⚖️**  

- `Cloud-Ready` 🌍 Can be deployed on **Netlify, Vercel, or Render**.  
- `Fast & Lightweight` ⚡ Optimized React app for **smooth performance**.  
- `Extensible` 🔧 Can be **integrated with additional AI models** in the future.  

<br/>

💡 **Phoenix AI** is your all-in-one **smart chatbot** that combines AI-powered conversations, **secure authentication**, and **seamless user experience** in a single platform. Enjoy chatting with intelligence! 🎉  

<br/>

# Technologies Used 👩🏻‍💻🤖  

- `@google/generative-ai` 🤖 Integration with **Gemini API** for AI-powered chat.  
- `appwrite` 🔐 Backend-as-a-Service for **authentication and database management**.  
- `framer-motion` 🎭 Smooth **animations and transitions** for an interactive UI.  
- `prop-types` ✅ Type-checking for **React components** to ensure reliability.  
- `react` ⚛️ Core **UI library** for building the frontend.  
- `react-dom` 🌐 Handles **React rendering** in the browser.  
- `react-helmet` 🎭 Manages **SEO-friendly meta tags** dynamically.  
- `react-markdown` 📄 Renders **Markdown content** inside React.  
- `react-router-dom` 🚏 Enables **client-side routing** for navigation.  
- `react-syntax-highlighter` 🖥️ Displays **code snippets** with syntax highlighting.  
- `remark-gfm` ✍️ Extends Markdown with **GitHub Flavored Markdown** (GFM) support.

# Installation Guide 🚀

## Clone 🏗️ the repository 🔍

### `For Windows` 🖥️

```bash
git clone https://github.com/Prakhar-002/PHOENIX.git
cd PHOENIX
```

### `For Linux and macOS` 🍏

```bash
git clone https://github.com/Prakhar-002/PHOENIX.git
cd PHOENIX
```

## Install 📥 dependencies 📦

```bash
npm install
```

## Create a `.env` ⚙️ file in the root directory and `add the following` ➡️

```ini
VITE_APPWRITE_PROJECT_ID='Your Appwrite Id'
VITE_GEMINI_API_KEY='Get you Api key from Gemini Api'
VITE_APPWRITE_DATABASE_ID='Make your own appwrite database and collections'
```

## Start ▶️ the application 🚀

```bash
npm run dev
```

## 🌍 Access the app in your browser at ➡️ 

🔗 [http://localhost:3000](http://localhost:3000)
